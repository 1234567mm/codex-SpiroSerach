# V25 Production Hardening And Reproducible Release Closure

Date: 2026-07-16

## Scope

V25 closes the V20–V25 roadmap with release evidence around the existing
SpiroSearch workflow. It adds deterministic release-hardening artifacts for
runtime support, migration/backward-read policy, security audit, performance
budgets, backup/restore recovery, and the release checklist.

Delivered:

- supported runtime profile with default dependencies and optional `ml`/`bo`
  extras kept out of the default runtime;
- migration policy report that fails closed on missing or mismatched schema
  references;
- security audit report for paths, secret-like payloads, command attribution,
  idempotency, and read-only surfaces;
- deterministic performance budget report for repository reads, project
  summaries, and viewer payload size;
- backup/restore recovery runbook evidence with hash/schema verification and no
  hidden credential requirement;
- reproducibility bundle and release checklist stub that records the final full
  gate command while explicitly avoiding external scientific-validation or lab
  dispatch claims.
